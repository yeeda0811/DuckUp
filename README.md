# DuckUp

這個專案是一個串接政府開放資料的網站，主要用於呈現流浪貓狗的相關資訊。我們使用了 Echarts.js 來製作各種圖表，以直觀且生動的方式呈現資料。
DuckUp 的由來是組員的暱稱，政府沒有流浪鴨子的資料。
